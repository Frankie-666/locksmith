{% trans "What is Locksmith?" %}

{% blocktrans %}Locksmith is a password storage service. We all have many accounts across many services. We wanted a simple, secure way to store the account credentials with an easy way to retrieve them.{% endblocktrans %}

{% trans "How secure is it?" %}

{% blocktrans %}We take security at the highest priority. Locksmith takes a slightly different approach from that of a typical web service. Most services will encrypt or hash your password so in the event that it is compromised, it is not easily readable. However, they will typically use a single, unique key to encrypt all data. With Locksmith, we do the same type of encryption however, every single user also has a unique "key" that is used as an additional randomizer (salt) during encryption. What this means is that even if your data is compromised, which we take great measure to prevent, every single user's data is completely unique making it even more difficult to decrypt.{% endblocktrans %}
